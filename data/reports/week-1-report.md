## Due Diligence Checklist

Before making any offer, confirm the following for each app:

**Technical**
- [ ] App still functions on Android 13 / 14
- [ ] Codebase language (Java / Kotlin / Flutter / React Native)
- [ ] Third-party API dependencies and their cost
- [ ] Any open-source licenses on the code

**Business**
- [ ] Monthly active users (ask the developer)
- [ ] Revenue: ad revenue + IAP breakdown
- [ ] Google Play Developer Console access — can it be transferred?
- [ ] Any pending policy violations or suspensions

**Legal**
- [ ] Developer owns the code fully (no agency or contractor disputes)
- [ ] No trademark conflicts on the app name
- [ ] User data handling — GDPR compliance if EU users exist
